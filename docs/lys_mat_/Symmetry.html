Symmetry module

class lys_mat.Symmetry.Symmetry(atoms, lattice)[source]

Bases: object

Symmetry class is used to calculate symmetry information of a crystal structure.

Parameters:

  • atoms (Atoms) – The Atoms object to get the symmetry information from.

  • lattice (CartesianLattice) – The CartesianLattice object to get the symmetry information from.

Example:

from lys_mat import Atom, Atoms, CartesianLattice, Symmetry
at1 = Atom("Na", (0, 0, 0))
at2 = Atom("Na", (0.5, 0.5, 0.5))
atoms = Atoms([at1, at2])
lat = CartesianLattice([4.2214, 4.2214, 4.2214, 90, 90, 90])
sym = Symmetry(atoms, lat)
print(sym.symmetryInfo())   #Symmetry: cubic Im-3m (No. 229), Point group: m-3m

getSymmetryOperations(pointGroup=False)[source]

Retrieve symmetry operations of the crystal structure.

This method uses spglib to obtain the symmetry operations for the crystal structure. If the pointGroup argument is set to True, it returns only the rotations that correspond to the point group by filtering out any translations. Otherwise, it returns both the rotations and translations.

Parameters:

pointGroup (bool) – If True, returns only symmetry operations corresponding to the point group.

Returns:

A list of rotation matrices if pointGroup is True. tuple: A tuple containing lists of rotation matrices and translation vectors if pointGroup is False.

Return type:

list

symmetryInfo()[source]

Retrieve symmetry information of the crystal structure.

This method uses spglib to obtain the symmetry dataset of the crystal structure and constructs a string containing the symmetry information. The string includes the crystal system, international symbol, space group number, and point group.

Returns:

A string representation of the symmetry information.

Return type:

str

Raises:

Exception – If the symmetry information cannot be determined.
